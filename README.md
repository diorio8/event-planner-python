# Event Planner Python 

Um organizador de eventos inteligente desenvolvido em Python para facilitar o planejamento de roles e encontros sociais.

##  Descrição do projeto
O event planner python é uma ferramenta criada para resolver a complexidade de divisão de gastos de forma precisa e organizada.

##  Funcionalidades (Em desenvolvimento)
- Cadastro de eventos.
- Lista de convidados.
- Controle de custos por pessoa.

##  Tecnologias
- Python 3
- Flet (Framework para interface grafica)
- Git e GitHub (Controle de versao)
